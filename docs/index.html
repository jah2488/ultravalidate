Refute, Don't Confirm

A claim survives only by surviving attack. Green tests prove the machine ran; they prove nothing about whether the conclusion is true. Before we report a result, we try to break it. Through that work we have come to value:

  • Refuting a claimoverconfirming it
  • The rawest sourceovera tidy summary
  • The weakest true statementoverthe strongest one that sounds provable
  • n, and a significance gateovera single confident sample
  • A named confoundovera clean story
  • A falsifier that already ranoverevidence that is "coming"

That is, while the items on the right are tempting, we trust the items on the left.

The five checks

1

Reconcile from source. Recompute the number from the rawest data on disk, never from a derived summary, and never from your own prior prose. A number you have not re-derived is unverified.

2

Fairness. Same conditions, budget, inputs, and n across everything compared. Name what differed; an unequal comparison is a confound wearing a result's clothes.

3

Power. What is n? One sample, one seed, one model is exploratory, not a finding. Does the claim's strength match the evidence's strength, and did it clear a gate?

4

Confound. What else could produce this besides the stated cause: a bug, a harness artifact, selection? Default to "there is a confound" until the obvious ones are ruled out.

5

Falsifier. What experiment would disprove this, and has it run? If it never executed, the claim is unproven. If the artifact is not on disk, the claim has outrun the evidence.

The verdicts

  • supported: the evidence holds up under attack.
  • exploratory-only: an early signal, too little data to conclude.
  • confounded: something else could explain the result, so the comparison does not prove the claimed cause. Name the something else.
  • unproven: the experiment that would prove it has not been run.
  • refuted: the evidence points the other way.

Report the weakest defensible restatement, with the uncertainty leading, not buried in a footnote. A clean "supported" is earned only after a genuine attempt to refute it failed.
